# Coffee App

A modern web application built with **React.js**, featuring multiple pages like Home, Menu, About, Contact, Blog, and Login.  
The project uses **React Router** for navigation and **Context API** for state management.

## 🚀 Features

- Multi-page navigation with React Router
- Global state management with Context API
- Dynamic blog pages with route parameters
- Deployable on Vercel/Netlify

## ⚡ Getting Started

```bash
npm install
npm run dev   # For Vite projects
# or
npm start     # For Create React App
```
